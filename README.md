## 🎯 Featured Games

### 🐍 Snake Game
Classic snake gameplay with modern visuals
- **Controls**: Arrow keys or WASD
- **Goal**: Eat food, grow longer, avoid hitting yourself
- **Features**: Score tracking, smooth animations

### 🧠 Memory Match
Test your memory with card matching
- **Controls**: Click to flip cards
- **Goal**: Find all matching pairs
- **Features**: Move counter, timer, difficulty levels

### 🔢 Math Blaster
Quick mental math challenges
- **Controls**: Click answers or use number keys
- **Goal**: Solve problems as fast as possible
- **Features**: Progressive difficulty, streak tracking

### ⌨️ Speed Typing
Test your typing speed and accuracy
- **Controls**: Type the displayed text
- **Goal**: Type as fast and accurately as possible
- **Features**: WPM tracking, accuracy percentage

### ⚡ Reaction Test
Test your reflexes
- **Controls**: Click when the circle turns green
- **Goal**: Click as fast as possible
- **Features**: Best time tracking, millisecond precision

### 🎯 Aim Trainer
Improve your clicking accuracy
- **Controls**: Click on targets as they appear
- **Goal**: Hit as many targets as possible
- **Features**: Accuracy tracking, moving targets

### 🌈 Color Match
Cognitive challenge game
- **Controls**: Click the color of the TEXT, not what it says
- **Goal**: Test your brain's processing speed
- **Features**: Lives system, streak tracking

### 🔤 Word Scramble
Unscramble letters to find words
- **Controls**: Type your answer
- **Goal**: Solve scrambled words across categories
- **Features**: Multiple categories, hint system

### 🎲 2048 Clone
Slide numbered tiles to reach 2048
- **Controls**: Arrow keys to move tiles
- **Goal**: Combine tiles to reach 2048
- **Features**: Undo functionality, best score tracking

### 🐦 Flappy Bird Style
Navigate through obstacles
- **Controls**: Click or spacebar to flap
- **Goal**: Fly as far as possible without hitting pipes
- **Features**: Physics-based movement, high score system

### 🌟 Asteroids Mini
Classic space shooter
- **Controls**: WASD/Arrow keys to move, Space to shoot
- **Goal**: Destroy asteroids and survive waves
- **Features**: Weapon heat system, progressive difficulty

### 🎨 Pixel Art Creator
Create beautiful pixel art
- **Controls**: Click and drag to draw
- **Goal**: Create amazing pixel artwork
- **Features**: Color palette, multiple tools, save functionality

### 🔍 Hidden Object
Find hidden objects in scenic environments
- **Controls**: Click on hidden objects
- **Goal**: Find all objects before time runs out
- **Features**: Zoom functionality, hint system, beautiful scenes

### 🌙 Tower Defense Mini
Strategic tower placement game
- **Controls**: Click to place towers
- **Goal**: Defend against waves of enemies
- **Features**: Multiple tower types, upgrade system

## 🛠️ Development

### Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/pranjal-coders/BitBlitz-Games.git
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

3. Navigate to `http://localhost:8000`

### Project Structure
```
BitBlitz-Games/
├── index.html          # Main HTML file with all games
├── README.md           # This file
└── assets/             # Additional assets (if any)
    ├── icons/          # Game icons
    └── screenshots/    # Game screenshots
```
